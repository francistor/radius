Overview
--------

This directory is where the Nokia AAA product saves backup files. These files could be
the result from the Setup program during upgrade or the installation of a Sample PolicyFlow.
In addition, when files saved from the SMT or MT, backup files are placed in the backup
directory.

Filename Format
-------------------

The name of the files in this directory are formatted using the original file name plus
the timestamp of when the original file was overwritten.

The timestamp portion is formatted as yyyy.MM.dd_HH.mm.ss

The modified time of the file is the last time it was saved.

Example file name format:

  radius_clients.xml.2014.08.25_18.18.17
